# VoronoiDiagram
An implementation of Voronoi diagram using Fortune's algorithm.

After find out all edges and vertices, extend size of map to fit the fareast vertex.
Then construct polygons by vertices.

## MonoGame Version
3.8.4

## Platform
Windows (Desktop GL)

## Usage
Press "N" key to generate next map.\
Press "P" key to pause the process.\
Press "R" key to redo the current map.
