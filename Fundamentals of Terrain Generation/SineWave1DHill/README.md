# SineWave1DHill
A simple implementation of 1D hill generation algorithm. Iterates a random value heightmap using sine wave interpolation.

## MonoGame Version
3.8.4

## Platform
Windows (Desktop GL)

## Usage
Press "N" key to generate next map.
