# PerlinNoise
An 2D version implementation of Perlin Noise with Fractal Brownian Motion.

The permutation table takes from Ken Perlin's [implementation](https://mrl.cs.nyu.edu/~perlin/noise/).\
The default frequency is set to 0.05, and the step size is 0.005.

## MonoGame Version
3.8.4

## Platform
Windows (Desktop GL)

## Usage
Press NumPad "+" key to increase the frequency.\
Press NumPad "-" key to decrease the frequency.\
Press "F" key to toggle whether fractal brownian motion is applied.
